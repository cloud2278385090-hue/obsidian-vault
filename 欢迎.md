# 欢迎来到你的第二大脑 🧠

> 由 [[灵境]] (OpenClaw AI) 搭建于 2026-05-04

---

## 🎯 快速开始

1. **收集** → 任何想法丢进 [[01 - Inbox/收集箱]]
2. **整理** → 定期把收集箱内容移到对应目录
3. **连接** → 用 `[[双链]]` 连接相关笔记
4. **回顾** → 每天写日记，每周回顾

## 📁 目录结构

| 目录 | 用途 |
|------|------|
| `00 - Daily` | 每日日记 |
| `01 - Inbox` | 收集箱（临时） |
| `02 - Projects` | 项目笔记 |
| `03 - Areas` | 责任领域 |
| `04 - Resources` | 资源库 |
| `05 - Archives` | 归档 |

## 🛠️ 已安装插件

- **dataview** — SQL式查询
- **obsidian-tasks-plugin** — 任务管理
- **quickadd** — 快速创建
- **periodic-notes** — 周期笔记
- **metaedit** — 属性编辑
- **metadata-menu** — 元菜单

- **obsidian-git** — Git备份

## 💡 常用命令

- `Ctrl/Cmd + O` — 快速切换
- `Ctrl/Cmd + N` — 新建笔记
- `[[` — 创建双链
- `Ctrl/Cmd + K` — 插入链接
